# Pulse Check User Journey

Your daily pulse on the world.

## User Types

1. Guests (logged out): Full site access, but cannot save articles or searches.
2. Registered Users (logged in): Full site access with the ability to save articles and searches. Local region will save to their profile.

## Functionality

All user types visiting the app will encounter the home page. The home page displays top news articles across the world and sub-category 'All' selected. The user can further filter the region by clicking 'US' or 'Local'. Selecting 'Local' will require the user to set their local region at the top of the page or if it has not been set after selecting 'Local', a popup will appear for the user to enter more information. Local can be defined by entering State, City, or Zip Code. State and City inputs will validate against a known list to ensure correct spelling, accepting both full state names and abbreviations. The current main category selected will be underlined. Guest users that set a local region should persist while revisiting.

Sub-categories below World, US, Local allow users to further filter for that region and topic. The sub-category selected will also be identified by an underline. To remove the sub-category filtering, the user will click All.

Article cards will load in paginated pages, displaying 3 x 3 articles at a time. The article card will have a photo from the article, title, publish date, source, and snippet of the article body. Articles are ordered by publish date. Logged in users can save articles by clicking the save flag to the right of the article. Clicking the article will send the user to the source website to read the article in a new tab.

If a search via category/sub-category or search function yields no results the user will encounter a message stating 'There are no articles matching this search criteria. Please try widening your search.

### Menu

Clicking the menu icon in the upper left corner will reveal the side bar. Options available will differ based on the user type.

1. Guests (logged out): About Us
2. Registered Users (logged in): About Us, Saved Articles, Saved Searches

### Search

Key words used in the search function will search ALL articles containing the key words in the title or description, disregarding the category/sub-category selection. Default sorted by publish date descending; clicking the active sort option again will toggle to ascending order. Save searches by clicking the Save Flag to the left of Search: {Search Criteria}.

### Registered User Features

1. Saved Articles: Article cards will load in paginated pages, displaying 3 x 3 articles at a time. The article card will have a photo from the article, title, publish date, source, and snippet of the article body. Articles can be unsaved by clicking the save flag and should disappear immediately. Default sorted by publish date in descending order; clicking the active sort option again will toggle to ascending order. Can also sort by the date they were saved.
2. Saved Searches: Re-running a saved search will conduct a new search. Searches can be unsaved by clicking the save flag and should disappear immediately. Ordered by the date they were saved in descending order; clicking the active sort option again will toggle to ascending order.

## Register an Account

Clicking the user icon in the upper right corner will bring a popup menu. The popup menu can be toggled between 'Login' (default) and 'Register'. A registered user will enter their email and password and click 'Submit' to login. Clicking the 'Password Reset' button will prompt the user to enter their email and they will receive an email to set a new password. An unregistered user can toggle to 'Register' and enter the following to create an account: First Name, Email, Password, and Confirm Password. Users registering for the first time will need to verify their email to activate their account. After logging in or verifying their email, a user will return to the previous page they were on. 

### Update Profile & Change Password

A logged in user will select 'Update Profile' in the drop down menu after clicking the user icon in the upper right corner. A popup menu will appear with 2 options to toggle from 'Profile' (default) and 'Password'. Under 'Profile', the user can update their First Name or Email. Under 'Password' a user can change their password by entering their current password, new password, and new password again to confirm.

### Logout

A logged in user will select 'Logout' in the drop down menu after clicking the user icon in the upper right corner.
